dotfiles

One command to set up a fresh macOS or Linux machine.

$ curl -fsSL https://doidor.github.io/.dotfiles/install.sh | bash
  1. Clones doidor/.dotfiles into ~/.dotfiles (or updates it).
  2. Runs setup.sh — installs Homebrew, zsh + oh-my-zsh, neovim, tmux, fzf, ripgrep and more.
  3. Symlinks every config with GNU Stow.
Source: bootstrap/install.sh. Review before piping to a shell.
